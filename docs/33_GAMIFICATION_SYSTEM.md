# The Argent Order

## Gamification System

Version: 1.0

Status: Canonical Behaviour Reinforcement System

---

# Purpose

Gamification exists to reinforce discipline, consistency, and responsibility.

Not entertainment.

Not addiction.

Not dopamine loops.

The system exists to make correct behaviour the easiest behaviour to continue.

---

# Core Principle

Gamification must reward virtue, not attention.

If a system rewards noise, it produces noise.

If it rewards discipline, it produces disciplined men.

---

# Primary Loop

```text id="loop"
Action

Tracking

Feedback

Progress

```

# Core Metrics System

The entire system is built around 5 core metrics.

---

## 1. Faith Score

Measures consistency in spiritual practice.

Includes:

* Prayer completion
* Scripture reading
* Mass attendance
* Rosary practice

---

## 2. Discipline Score

Measures self-control and execution.

Includes:

* Wake time consistency
* Exercise completion
* Habit adherence
* Digital discipline

---

## 3. Brotherhood Score

Measures engagement with community.

Includes:

* Pod attendance
* Check-ins
* Accountability responses
* Support given to others

---

## 4. Builder Score

Measures output and creation.

Includes:

* Projects shipped
* Work sessions completed
* Deliverables created
* Progress logs

---

## 5. Truth Score

Measures honesty and self-awareness.

Includes:

* Daily reflection completion
* Failure logging
* Weekly review honesty
* Avoidance detection

---

# SCORING PRINCIPLE

Scores are not for comparison.

They are for clarity.

---

Bad usage:

* Ranking men against each other
* Ego inflation
* Competitive superiority

---

Correct usage:

* Identifying weakness
* Tracking growth
* Guiding correction
