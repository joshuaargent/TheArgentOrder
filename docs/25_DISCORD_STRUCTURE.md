# The Argent Order

## Discord Structure

Version: 1.0

Status: Canonical Community Architecture

---

# Purpose

Discord is not the product.

Discord is the battlefield.

It handles:

* Brotherhood
* Accountability
* Communication
* Enforcement
* Culture formation

The Portal handles structure.

Discord handles behaviour.

---

# Core Design Principle

No chaos.

No entertainment-first channels.

No infinite scrolling.

Every channel must exist for:

```text id="core1"
Action
Accountability
Formation
Leadership
Execution
```
