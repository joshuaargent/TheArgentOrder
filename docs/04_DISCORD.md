# The Argent Order
## Discord Specification

Version: 1.0

Status: Canonical Community Architecture

---

# Purpose

Discord is the Brotherhood Layer.

Discord exists to:

- Build relationships
- Create accountability
- Facilitate discussion
- Coordinate campaigns
- Strengthen culture

Discord does not exist to:

- Replace the Portal
- Replace formation
- Become a social media feed
- Maximize screen time
- Encourage endless conversation

The Portal is where men improve.

Discord is where brothers gather.

Discord is not the product.

Discord is the battlefield.

It handles:

* Brotherhood
* Accountability
* Communication
* Enforcement
* Culture formation

The Portal handles structure.

Discord handles behaviour.

---

# Core Philosophy

Discord should feel like:

A monastery
+
A military academy
+
A builder community

The atmosphere should be:

- Serious but welcoming
- Disciplined but human
- Structured but not sterile

The goal is not activity.

The goal is meaningful activity.

---

# Discord Principles

## Principle 1

Quality over quantity.

Ten meaningful messages are worth more than one thousand pointless messages.

---

## Principle 2

Brotherhood over entertainment.

The Order is not a gaming server.

The Order is not a meme server.

The Order is not a social club.

---

## Principle 3

Formation over engagement.

Engagement is not success.

Transformation is success.

---

## Principle 4

Action over discussion.

Discussion should produce action.

Action should produce growth.
